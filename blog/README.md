# 博客展示系统

基于 Vue 3 + TypeScript + Element Plus + Supabase 开发的博客展示系统。

## 功能特性

- ✅ 用户登录/注册
- ✅ 文章列表展示（卡片式布局）
- ✅ 文章详情页（支持 Markdown 渲染）
  - 支持 `**加粗**`、`*斜体*`、`~~删除线~~`
  - 支持 `!!隐藏文字!!`（鼠标悬停显示）
  - 支持代码高亮、图片、链接等
- ✅ 按分类筛选
- ✅ 按标签筛选
- ✅ 文章搜索（标题和内容）
- ✅ 暗色/白色主题切换
- ✅ 响应式设计
- ✅ 分页功能

## 安装依赖

```bash
npm install
```

## 配置环境变量

将 `env.txt` 文件重命名为 `.env`（已包含你的 Supabase 配置）

## 运行开发服务器

```bash
npm run dev
```

访问: http://localhost:3001

## 构建生产版本

```bash
npm run build
```

## 页面说明

### 首页 (/)
- 展示所有已发布的文章
- 支持按标签筛选
- 卡片式布局，显示文章封面、标题、摘要、分类、标签等信息

### 文章详情页 (/post/:id)
- 显示完整的文章内容（Markdown 渲染）
- 显示文章元信息（分类、标签、发布时间）
- 支持特殊 Markdown 语法

### 分类页 (/category/:slug)
- 显示指定分类下的所有文章

### 标签页 (/tag/:slug)
- 显示指定标签下的所有文章

### 搜索页 (/search?q=keyword)
- 根据关键词搜索文章标题和内容

### 登录页 (/login)
- 用户登录/注册

## 特色功能

### Markdown 支持

文章内容支持以下特殊语法：

- `**加粗**` → **加粗**
- `*斜体*` → *斜体*
- `~~删除线~~` → ~~删除线~~
- `!!隐藏文字!!` → 鼠标悬停可见
- 代码块、链接、图片等标准 Markdown 语法

### 主题切换

点击导航栏的主题按钮即可切换暗色/白色主题，主题偏好会自动保存。

### 响应式设计

支持桌面端和移动端访问，自适应不同屏幕尺寸。

## 与后台管理系统配合

该博客展示系统需要配合 `admin` 目录下的后台管理系统使用：

1. 在后台管理系统中创建分类、标签
2. 在后台管理系统中创建和发布文章
3. 在博客展示系统中查看已发布的文章

两个系统共享同一个 Supabase 数据库。
